# Project Shoping
## Development
### Project Setup
```bash
https://git.arkademy.com/tatasfachrul/project-shopping.git
npm install
```

### Start development
```bash
npm start
```

### End Point Setup
Edit and add your Host for API in `setServer.js`
```bash
export const ip='http://<API Host>'
```

## Build
### Project Build
```bash
npm run build
```
